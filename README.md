# 🧮 Calculadora em Python

Uma calculadora simples feita em Python que realiza operações básicas diretamente pelo terminal.

---

## 🚀 Funcionalidades

- ➕ Adição
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- ⚠️ Tratamento de divisão por zero

---

## 💻 Como usar

### ▶️ Rodando com Python
1. Baixe o arquivo `calculadora.exe`
2. Execute, pronto!
